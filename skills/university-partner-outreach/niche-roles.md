# Niche → role targeting

Open only the row(s) that match the university org. Do not load this whole file into reasoning as a checklist for every company.

Partner / university / campus owners stay useful across niches. Primary cold titles should match the org.

| Org niche | Primary roles | Usually skip unless user asks |
|-----------|---------------|-------------------------------|
| **Technical / SWE / CS**, hackathon, robotics | TPMs, eng managers/directors (relevant stack), DevRel, developer advocates, university recruiting (tech), solutions eng managers | Brand marketers, generic sales, finance |
| **Product / PM club** | PMs/directors, PMM, product ops, early careers (PM) | Deep infra eng with no product surface |
| **Strategy / management consulting** | Strategy & ops, firm-side consultants, BD, PMM, program managers, university recruiting (non-tech) | IC software engineers, deep eng managers |
| **Marketing / communications** | Marketing managers/directors, brand, growth, content, PMM, campus marketing | Pure eng delivery, tax/finance |
| **Finance / investment** | Corp dev, IR (if campus/programs), FP&A case-comp sponsors, university recruiting (finance) | Eng managers, DevRel |
| **Design / HCI / creative** | Design managers/directors, design systems, UX research managers, creative brand | Backend eng managers |
| **Data / analytics / AI (applied)** | DS/ML managers, analytics leads, AI PMs, applied science managers, ML DevRel | Quota AE with no program remit |
| **Entrepreneurship / VC** | Startup program leads, venture/platform partners, campus innovation, bizdev | IC eng |
| **Healthcare / bio / pre-med** | Medical affairs education, clinical partnerships, university relations (health), relevant product | Unrelated consumer brand marketers |
| **Policy / public interest / law** | Public policy, gov affairs education, legal community programs, university relations | Product eng |

## Ask-type overlays (stack on niche)

| Ask type | Prefer |
|----------|--------|
| Campus / talent / recruiting | University recruiting, early careers, TA partners, academic alliances, DevRel education |
| Partnership / ecosystem / sponsorship | Partners, alliances, channels, ISV/GSI, partner marketing, community/education sponsors |
| Client / commercial intro | Partner sales, SE mgrs, CS directors (not pure finance) |
| Speakers / workshops | Niche practitioners + program owners (EM/TPM for SWE; PM/PMM for consulting/marketing) |

If niche is mixed after the profile, ask one clarifying question. If still unclear, use a balanced mix and say so in the summary.
